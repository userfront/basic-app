Userfront demo
Be sure to add your Project ID to the Userfront script on each page.
These tools are included with your Userfront project:

Signup form

Allows a new user to sign up for your project.

When a user signs up, they are redirected to the URL of your choice (default is /dashboard). They also receive a welcome email with a link to confirm their email address.

Login form

Allows an existing user to log into your project.

When a user logs in, they are redirected to the URL of your choice (default is /dashboard). In addition to logging in directly, users can request a login link or password reset email.

Password reset form

Allows an existing user to reset their password.

When users request a password reset email, they are directed to the URL of your choice (default is /reset). This form is used to send the password reset email and also to reset the password.

Logout button

Allows a logged in user to log out.

When users log out, they are directed to the URL of your choice (default is /login).
